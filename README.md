# 🤟 Indian Sign Language Recognition using CNN

A deep learning project that recognizes **Indian Sign Language (ISL)** hand gestures using a Convolutional Neural Network (CNN). The model classifies hand gesture images into their corresponding sign language classes.

---

## 📖 Project Overview

This project demonstrates how Convolutional Neural Networks (CNNs) can be used for image classification in computer vision. The model is trained on an Indian Sign Language dataset and predicts the corresponding hand sign from an input image.

---

## 🚀 Features

- Hand gesture recognition using CNN
- Image preprocessing with OpenCV
- Image classification using TensorFlow/Keras
- Model training and evaluation
- Prediction on new images
- Implemented in Jupyter Notebook

## ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/janathiru25/Indian-sign-language-recognition.git
```

Move into the project folder:

```bash
cd Indian-sign-language-recognition
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Start Jupyter Notebook:

